# 2. 好用的快捷键

# 2.1. 通用编辑

 说明 | 快捷键
-- | --
 列模式 | 按住 `Shift + Alt`
 复制行 | `Shift + Alt + <Up/>Down>`
 删除行 | `Ctrl + Shift + K`
 移动行 | `Alt + <Up/Down>`
 连接行 | `Ctrl + J`
 
<br>

# 2.2. Markdown

 说明 | 快捷键
-- | --
 粗体 | `Ctrl + B`
 斜体 | `Ctrl + I`
 代码块 | `Ctrl-M + Ctrl-C`
 行内代码块 | `Ctrl-M + Ctrl-I`

<br>

# 2.3. C/C++ Tag

 说明 | 快捷键
-- | --
 跳转到定义 | `F12`
 调到引用 | `Shift + F12`
 全局symbol | `Ctrl-T`
 文件内symbol | `Ctrl-Shift+O`
